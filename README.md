# 📚 Learning Path Generator

Generador de rutas de aprendizaje impulsado por IA que crea planes de estudio personalizados a partir de un objetivo de aprendizaje definido por el usuario.

## ✨ Features

- Generación automática de rutas de aprendizaje.
- Planes estructurados por módulos y etapas.
- Integración con modelos de IA.
- Interfaz intuitiva y responsive.
- Personalización según objetivos específicos.
- Arquitectura escalable para futuras integraciones.

## 🛠️ Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS

### Backend
- Node.js
- Express

### AI
- OpenAI API

### Database
- PostgreSQL / Supabase
